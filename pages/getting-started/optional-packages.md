# Installing Optional Packages

The Meadow plugin allows you to install optional packages that extend Meadow's capabilities.
These include:

- **Experience Builder** - A toolkit for creating complex interactive experiences in Meadow.
- **AR Extensions** - Adds additional AR functionallity such as image tracking, object tracking, and more.

Follow the steps bellow to install these packages:

1. In Unity, on the top menu bar, click **Meadow**.
![alt_text](../images/optional-packages-1.webp)

2. Select **Optional Packages**.
![alt_text](../images/optional-packages-2.webp)

3. Check the packages you want to install, then click **Install**.
![alt_text](../images/optional-packages-3.webp)
